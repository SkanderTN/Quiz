QUIZ-GAME

Simple full-stack Quiz Game.

Backend: Node / Express / MongoDB
Frontend: Angular (client + server bundles)

Prerequisites
- Node.js and npm
- MongoDB running and accessible

How to run (final, minimal)

1) Backend

cd backend
npm install
Ensure `backend/.env` contains a valid MONGO_URI (e.g. mongodb://localhost:27017/Quiz-Game) and PORT if desired.
Start the backend:
node app.js

2) Frontend (development)

cd frontend
npm install
Start the Angular dev server:
npm start

Open the app at: http://localhost:4200/
